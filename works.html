Full Stack Development

I spent 3 months learning and working on a project from end-to-end. I made the backend in Java, with a Mongo database, using Spring framework.

The frontend is made in JavaScript, using React Native for the mobile app. We used React for the website.

Both the mobile app and the website queried the database securely, and required tokens or a new login to access secure resources.

Spring Security & JWT

I added security to our project using Spring Security to create JWT access tokens for the user, as well as implementing JWT refresh tokens for a more user-friendly experience. I learned about OAuth and how to create a secure service for the user.

Projects from Freshman Year

In addition, I learned about multi-threading and sorting algorithms. I learned about algorithmic complexity and how to find it for algorithms. Projects from Sophomore Year

In my sophomore year of college, I learned about the importance of memory management with C, as well as basic low-level MIPS Assembly programming. I learned OCaml, Ruby, Python, and Rust. We created our own basic compilers in class.

From May 2018 to August 2019, I worked with Interclypse and Sofiac to write software predominantly in Java. We use Git, Maven, MongoDB, Spring, and Docker to build this project. In addition, I used React, React-Native, Expo, and Spring Security to create secure full-stack projects and mobile apps.

I spent 6 months as a contracted intern with the U.S. Food and Drug Administration working with a team researching noroviruses.
